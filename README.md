# STATIC-WEBSITE-HOSTING-AWS-TERRAFORM

## 📌 Overview

A Terraform-based Infrastructure as Code (IaC) project that automates the deployment of a static website on AWS using Amazon S3 Static Website Hosting.

---

## 🏗️ Architecture

```text
User → S3 Static Website Hosting → Website Endpoint
```

---

## 🛠 Technologies Used

- AWS S3
- Terraform
- Git
- GitHub

---

## 🚀 Features

- Automated S3 website deployment
- Infrastructure managed through Terraform
- Public website hosting
- Repeatable and scalable infrastructure

---


## 📚 Learning Outcomes

- AWS S3 Static Website Hosting
- Infrastructure as Code (IaC)
- Terraform Resource Management
- Cloud Infrastructure Automation
